# 🛡️ 阶段七：运行时监控 (Runtime Detect)
**目标**: 如果坏人已经进来了，我要能立刻发现他在干坏事。

> **👶 小白概念站**:
> *   **运行时**: 程序正在跑的时候。
> *   **异常行为**: 比如 Web 服务器突然启动了一个 Shell，或者修改了 `/etc/shadow` 文件。

## 📝 学习任务

* [ ] **部署“保安” (Falco)**
  * 操作: 使用 Helm 安装 Falco。
* [ ] **触发报警 (实战验证)**
  * **场景 1**: 在容器里运行 `cat /etc/shadow` (读取密码文件)。
  * **场景 2**: 在容器里运行 `bash -i` (反弹 Shell 的前兆)。
  * 验证: 查看 Falco 的日志输出，确认它抓到了这些行为。
