# ⚔️ 阶段九：红队演练 (Red Teaming)
**目标**: 像黑客一样思考，攻击自己的集群，找到漏洞。

> **👶 小白概念站**:
> *   **提权**: 用普通用户的身份进去，想办法变成管理员 (Root)。
> *   **逃逸**: 从 Docker 容器里跳出来，控制宿主机。

## 📝 学习任务

* [ ] **特权容器逃逸**
  * **实验**: 创建一个 `privileged: true` 的 Pod。
  * 攻击: 尝试挂载宿主机的磁盘，读取宿主机的各个文件。
* [ ] **使用自动化工具**
  * **工具**: CDK (Container Drill Kits) 或 Peirates。
  * 操作: 在 Pod 里运行工具，自动扫描有哪些漏洞可以利用。

## 🏆 里程碑练习：越狱

**任务**: 部署一个拥有 `SYS_ADMIN` 能力的 Pod，利用 cgroup `release_agent` 逃逸技术，在宿主机上创建一个文件 `/root/pwned`。

**✅ 通关标准**: `ls /root/pwned` 在宿主机上执行成功。
